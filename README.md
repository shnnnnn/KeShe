个人任务管理与提醒系统
功能要求：开发一个个人任务管理与提醒系统，帮助用户有效地管理日常任务、设置提醒，并跟踪任务完成进度。
基本功能：
1.	任务管理：用户可以添加、编辑、删除任务，并为每个任务设置到期时间、优先级等信息。
2.	任务分类：支持将任务按不同类别（如工作、学习、生活等）进行分类管理。
3.	提醒功能：系统在任务到期前通过通知或弹窗提醒用户。
4.	任务进度跟踪：用户可以标记任务为“进行中”、“已完成”或“未完成”，并查看任务的历史记录。
扩展功能 ：
1.	定期任务：支持设置定期任务（如每天、每周、每月等），自动生成和提醒。
2.	任务共享与协作：用户可以将任务分享给其他用户，共同完成任务。

技术要求：
•	安卓原生开发，使用SQLite数据库进行任务数据存储。
•	通知功能使用 Android 的通知系统进行任务提醒。
